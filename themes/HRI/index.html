Human-Robot Interaction

I work on using human experiences to improve semantic understanding of the environment for mobile robots (Rosa et al., 2018),(Rosa et al., 2018) .

I study interpretable ways to learn sensor fusion strategies in deep VIO frameworks, for integrating novel sensor modalities such as millimeter wave radar and thermal imaging into a unified framework for first responders (Chen et al., 2022), (Saputra et al., 2020), (Lu et al., 2020), (Chen et al., 2019) .
